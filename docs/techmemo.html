windowsにおけるvim設定

バイナリインストールした場合

ユ―ザの環境変数にvim.exeのパスを追加する。
→windows terminalから実行できるようになる。

vimrc設定

C:\Users\（ユーザ名）\_vimrcを作成、以下のテキストを記入し保存する。
"=========================================
" Basic Vim Settings for Windows
"=========================================
" Enable syntax highlighting
syntax on
colorscheme murphy

" Set line numbers
set number

" Set color
set termguicolors

" Show current line
set cursorline

" Disable creation of backup and swap files
set nobackup
set nowritebackup
set noswapfile

" tab key settings
set expandtab
set tabstop=4

linux全般

・ユーザを新規追加

・ユーザを既存グループに追加
$ usermod -aG [既存グループ名] [ユーザ名]

git

初期設定

新規に開発環境を準備する場合、以下コマンドを実行し、gitconfigに設定する。
$ git config --global user.name [任意のユーザ名]
$ git config --global user.email [任意のメールアドレス]
$ git config --global alias.br branch
$ git config --global alias.st status
$ git config --global alias.la "log --all --graph --oneline"
